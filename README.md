## How to

Running the application

```shell
# Install deps
npm install -g typescript@1.8.10
npm install -g typings
npm install -g gulp
npm install -g lite-server
npm install -g rimraf

# fresh install
npm run clean-install

# build and run
npm run build

# just run
npm start
```
